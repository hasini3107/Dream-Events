# 📖 Overview

Dream Events is a full-stack event management platform designed to simplify event planning and booking. The platform enables users to browse event services, explore different categories, and manage bookings through an intuitive web interface.

Whether it's a wedding, birthday celebration, corporate event, or special occasion, Dream Events helps users organize memorable experiences with ease.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/hasini3107/Dream-Events.git
```

Go to the project directory

```bash
cd Dream-Events
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Flask application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000/
```
